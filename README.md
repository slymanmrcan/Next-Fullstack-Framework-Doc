# Next.js Full Stack Framework Dokümanı

Bu repo, GitHub Pages'te statik olarak yayınlanabilir tek sayfalık bir doküman içerir. 
live at https://slymanmrcan.github.io/Next-Fullstack-Framework-Doc/
Doküman Next.js tam yığın yaklaşımını, Prisma/ORM kullanımı, API route tasarımı, Swagger/OpenAPI entegrasyonu, auth seçenekleri, Docker build ve dosya tabanlı yönlendirme gibi konuları kapsar.

## İçerik
- `index.html`: Doküman sayfası (koyu tema, menülü, bölüm bölüm).
- Auth, middleware, lifecycle, DB seçenekleri, Docker örneği, file-system routing ve daha fazlası bu dosyanın içindedir.

## Kullanım
1) Tarayıcıda aç: `open index.html`
2) GitHub Pages’e koymak için repo root’una commit/push et ve Pages’i `main` branch / `/` (root) olarak ayarla.

## Notlar
- Yeni bölümler eklemek için doğrudan `index.html` içindeki ilgili section’ı kopyalayıp düzenleyebilirsin.
- Docker örneği multi-stage ve Prisma generate adımı içerir; kendi paket yöneticin ve komutlarına göre uyarlayın.
